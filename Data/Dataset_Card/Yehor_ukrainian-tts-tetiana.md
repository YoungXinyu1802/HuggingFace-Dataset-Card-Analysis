# Voice TETIANA (female)

License: [Apache 2.0](https://github.com/egorsmkv/ukrainian-tts-datasets/blob/main/LICENSE)

## Features

- Quality: high
- Duration: 8h
- Audio formats: OPUS/WAV
- Text format: JSONL (a `metadata.jsonl` file)
- Frequency: 16000/22050/48000 Hz

## Original version
