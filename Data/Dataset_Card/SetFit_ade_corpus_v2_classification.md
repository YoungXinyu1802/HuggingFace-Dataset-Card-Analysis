# ADE-Corpus-V2 Dataset: Adverse Drug Reaction Data.
This is a dataset for classification if a sentence is ADE-related (True) or not (False).

**Train size: 17,637**

**Test size: 5,879**

[Source dataset](https://huggingface.co/datasets/ade_corpus_v2)
